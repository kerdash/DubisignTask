# DubiSign Task

### Installing

```
1. Run `composer install`
2. Copy .env.example to .env
3. Configure the database settings in .env
4. Run `php artisan migrate`
5. Run `php artisan serve`
6. Visit http://localhost:8000 in your browser
```
